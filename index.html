⚙️ Βασικές Ενέργειες

Επανεκκίνηση (Restart)

Βασική ενέργεια αντιμετώπισης προβλημάτων

Η διαδικασία κλεισίματος και επανεκκίνησης του υπολογιστή ή μιας εφαρμογής. Διακόπτει όλες τις τρέχουσες διεργασίες και τις ξεκινά από την αρχή. Είναι συχνά η πρώτη και απλούστερη λύση σε πολλά τεχνικά προβλήματα.

Πότε να το χρησιμοποιήσεις: Όταν μια εφαρμογή δεν ανταποκρίνεται, όταν ο υπολογιστής είναι αργός, όταν εμφανίζεται άγνωστο σφάλμα.

Format (Μορφοποίηση)

Ακραία ενέργεια — χρήση μόνο ως έσχατη λύση

Η πλήρης διαγραφή όλων των δεδομένων από τον σκληρό δίσκο και επανεγκατάσταση του λειτουργικού συστήματος. Λύνει πολλά προβλήματα αλλά διαγράφει τα πάντα — αρχεία, εφαρμογές, ρυθμίσεις.

⚠️ Προσοχή: Πριν κάνεις format, φρόντισε να έχεις αντίγραφο ασφαλείας (backup) όλων των αρχείων σου!

Force Quit (Αναγκαστικό Κλείσιμο)

Κλείσιμο εφαρμογής που έχει παγώσει

Η αναγκαστική διακοπή μιας εφαρμογής που δεν ανταποκρίνεται και δεν κλείνει κανονικά. Στα Windows γίνεται μέσω Task Manager (Ctrl+Alt+Del) και επιλογή "End Task".

Πώς: Ctrl + Alt + Del → Task Manager → κλικ στην εφαρμογή → End Task

Task Manager (Διαχείριση Εργασιών)

Εργαλείο παρακολούθησης του συστήματος

Εργαλείο των Windows που δείχνει όλες τις τρέχουσες εφαρμογές και διεργασίες, καθώς και πόσους πόρους (CPU, RAM) καταναλώνουν. Βοηθά να εντοπίσεις ποια εφαρμογή επιβαρύνει τον υπολογιστή.

Πώς να το ανοίξεις: Ctrl + Alt + Del → Task Manager, ή δεξί κλικ στη γραμμή εργασιών → Task Manager

Router / Modem

Συσκευή σύνδεσης στο internet

Το router/modem είναι η συσκευή που συνδέει τον υπολογιστή σου με το internet. Όταν χάνεις τη σύνδεση, η επανεκκίνηση του router είναι συχνά η πρώτη λύση.

Πώς να κάνεις restart: Αποσύνδεσε το καλώδιο ρεύματος, περίμενε 30 δευτερόλεπτα, ξανασύνδεσε.

WiFi

Ασύρματη σύνδεση στο internet

Τεχνολογία ασύρματης σύνδεσης που επιτρέπει στις συσκευές να συνδέονται στο internet χωρίς καλώδιο. Προβλήματα WiFi μπορεί να οφείλονται σε απόσταση από το router, εμπόδια ή πρόβλημα στον πάροχο.

Διάγνωση: Αν άλλες συσκευές συνδέονται κανονικά, το πρόβλημα είναι στη συσκευή σου και όχι στο router.

CPU (Επεξεργαστής)

Ο "εγκέφαλος" του υπολογιστή

Η CPU (Central Processing Unit) είναι ο κεντρικός επεξεργαστής του υπολογιστή — εκτελεί όλες τις υπολογιστικές λειτουργίες. Όταν η χρήση CPU φτάνει κοντά στο 100%, ο υπολογιστής γίνεται εξαιρετικά αργός γιατί δεν μπορεί να ανταποκριθεί σε νέες εργασίες.

Πώς να την ελέγξεις: Task Manager → καρτέλα "Performance" → CPU. Αν είναι πάνω από 80-90% συνεχώς, κάποια εφαρμογή την επιβαρύνει.

Temporary Files (Προσωρινά Αρχεία)

Αρχεία που δημιουργούνται αυτόματα κατά τη χρήση

Αρχεία που δημιουργούν αυτόματα τα Windows και οι εφαρμογές κατά τη λειτουργία τους. Συσσωρεύονται με τον καιρό και καταλαμβάνουν χώρο στον δίσκο, επιβραδύνοντας τον υπολογιστή.

Πώς να τα διαγράψεις: Πληκτρολόγησε %temp% στην αναζήτηση των Windows → επίλεξε όλα → διέγραψε.

RAM (Μνήμη)

Προσωρινή μνήμη του υπολογιστή

Η RAM είναι η "εργατική" μνήμη του υπολογιστή που χρησιμοποιείται για να τρέχουν οι εφαρμογές. Όταν η RAM είναι γεμάτη, ο υπολογιστής γίνεται αργός και εφαρμογές μπορεί να παγώνουν.

Σημείωση: Η RAM διαφέρει από τον σκληρό δίσκο — ο δίσκος αποθηκεύει μόνιμα, η RAM είναι προσωρινή.
🔒 Ασφάλεια

Firewall (Τείχος Προστασίας)

Εργαλείο ασφάλειας — ΜΗΝ το απενεργοποιείς!

Το firewall είναι ένα σύστημα ασφαλείας που ελέγχει και φιλτράρει την κυκλοφορία δικτύου, προστατεύοντας τον υπολογιστή από μη εξουσιοδοτημένη πρόσβαση και κακόβουλο λογισμικό.

⚠️ Σημαντικό: Ποτέ μην απενεργοποιείς το firewall για να λύσεις ένα πρόβλημα — αφήνεις τον υπολογιστή εκτεθειμένο σε επιθέσεις.

Driver (Πρόγραμμα Οδήγησης)

Λογισμικό επικοινωνίας συσκευής

Ένα driver είναι ένα μικρό πρόγραμμα που επιτρέπει στα Windows να "μιλούν" με μια συσκευή (εκτυπωτή, κάμερα, κάρτα δικτύου). Παλιά ή κατεστραμμένα drivers μπορεί να προκαλούν δυσλειτουργίες.

Πώς να ενημερώσεις: Διαχείριση Συσκευών (Device Manager) → δεξί κλικ στη συσκευή → Update driver
🔧 Λογισμικό

Uninstall / Reinstall (Απεγκατάσταση / Επανεγκατάσταση)

Λύση για προβλήματα εφαρμογών

Η πλήρης διαγραφή μιας εφαρμογής (uninstall) και η εκ νέου εγκατάστασή της (reinstall). Χρησιμοποιείται όταν μια εφαρμογή έχει κατεστραμμένα αρχεία και δεν λειτουργεί σωστά παρά την επανεκκίνηση και ενημέρωση.

Πώς: Ρυθμίσεις → Εφαρμογές → βρες την εφαρμογή → Κατάργηση εγκατάστασης → κατέβασέ την ξανά.

Update (Ενημέρωση)

Σημαντική για ασφάλεια και λειτουργικότητα

Οι ενημερώσεις διορθώνουν σφάλματα, βελτιώνουν την απόδοση και κλείνουν κενά ασφαλείας. Παλιές εκδόσεις εφαρμογών συχνά παρουσιάζουν προβλήματα που λύνονται με απλή ενημέρωση.

Συμβουλή: Κράτα πάντα ενημερωμένα τα Windows και τις εφαρμογές σου για βέλτιστη λειτουργία.
